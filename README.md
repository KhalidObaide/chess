## Chess
Simple chess game built with the help of SDL.

All the basic rules & features are implemented:
* Piece Movements
* Special Moves ( En-Passant & Castling )
* Pawn Promotions
* Check, Checkmate and Stalemate validation
* Chess board theme switcher
* Board Flip
* Restart Trigger

## Setup & Usage:
1. `make all`
2. `make run`
